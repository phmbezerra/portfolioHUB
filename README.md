### Principais pontos:
- interface moderna e limpa;
- responsividade;
- boa hierarquia visual;
- organização de seções para apresentação de produto.

## Organizador de Arquivos
Script em Python para organizar arquivos automaticamente por extensão, criando subpastas e melhorando a produtividade.

### Principais pontos:
- automação de tarefas repetitivas;
- manipulação de arquivos;
- organização lógica;
- aplicação prática fora do contexto acadêmico.

### Principais pontos:

- uso de banco de dados SQLite;
- estruturação de pequenas aplicações;
- foco em lógica e persistência.
